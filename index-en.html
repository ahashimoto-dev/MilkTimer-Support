MilkTimer

Breastfeeding Timer App

About the App

MilkTimer is a breastfeeding timer app designed for mothers with newborns. It provides a simple and easy-to-use timer function to help ensure equal feeding time on both breasts.

Features an eye-friendly light pink design that's easy to see even during nighttime feedings. The simple interface allows for easy one-handed operation while nursing.

How to Use

1. Set Time

  • Set feeding time (default: 5 minutes)
  • Set interval time (default: 20 seconds)

2. Optional Settings

  • Left/Right Display: Choose whether to show which breast during feeding
  • Starting Side: Select whether to start with right or left breast (only when L/R display is ON)
  • Notification Sound: Turn notification sound ON/OFF when time ends

3. Start

  • Tap the "Start" button

4. Stop
